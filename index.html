⚙️ Setup Steps

1
Create a Google SheetAdd headers in Row 1: Token | Name | Phone | Email | Hospital | Dept | City | Date | Slot | Booked At
2
Create an Apps Script Web AppIn your Sheet → Extensions → Apps Script → paste the doPost() function from credentials.js → Deploy as Web App (Anyone can access) → copy the URL
3
Paste the URL in script.jsOpen script.js → set GOOGLE_APPS_SCRIPT_URL = 'https://script.google.com/macros/s/…/exec' at the top
4
Done — no backend needed!Every new booking now auto-saves a row to your Google Sheet instantly, directly from the browser
✅ Auto-save is already wired in. Every booking calls saveToGoogleSheets() and triggerMakeWebhook() automatically. Just set your URL in script.js and it starts logging instantly — no Node.js server required.
